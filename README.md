# BigBrotr Presentation

Slide deck for [BigBrotr](https://github.com/BigBrotr/bigbrotr) — a modular Nostr data archiving and monitoring system.

Built with [Slidev](https://sli.dev/).

## Development

```bash
npm install
npm run dev       # Start dev server with live reload
npm run build     # Build static SPA for deployment
npm run export    # Export to PDF
```

## Deployment

Deployed automatically to [Vercel](https://vercel.com/) on push to `main`.
